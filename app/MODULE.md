# app 模块说明

## 1. 模块职责

`app` 是当前项目的应用入口模块，负责：

- Android Application 初始化
- 依赖注入启动
- Activity 入口承载
- 各业务模块的装配

## 2. 当前包含内容

- `QuickTranslateApplication`
  - 启动 Koin
- `MainActivity`
  - 应用主入口 Activity
- `di/AppModule.kt`
  - 应用级依赖注入装配

## 3. 当前实现状态

目前该模块完成了：

- Compose 应用入口接入
- 业务模块装配骨架
- Koin 初始化

目前该模块尚未完成：

- 文件导入
- 导航体系
- 详细页面路由编排

## 4. 对外暴露能力

当前该模块主要提供应用运行入口，不对外暴露业务组件接口。

## 5. 后续维护注意事项

- 若 `app` 模块后续仅保留入口职责，应持续把业务代码迁移到独立模块
- 若新增全局初始化逻辑，需要同步更新本文件
- 若调整入口页面、依赖注入装配或导航职责，需要同步更新本文件
